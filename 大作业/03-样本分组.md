# Project Log: 2025-12-24
## 任务：样本分组 (Sample Grouping)

### 1. 目标
解析 `GSE277909_ID_Age.txt` 文件，建立样本 ID 到年龄分组的映射关系。
- Group1: Age < 30
- Group2: 30 <= Age <= 50
- Group3: Age > 50

### 2. Pains (难点)
- **文件 IO**: C 语言读取文本需要处理表头 (Header) 跳过的问题。
- **字符串处理**: 样本 ID (如 `1404DNA01`) 是字符串，不能直接作为数组下标，需要设计结构体 (`struct`) 来存储。
- **内存管理**: 样本数量未知，需要使用动态扩容 (本次采用 `malloc` 一次性分配)。

### 3. Gains (收获)
- **结构化思维**: 定义了 `Sample` 结构体，将 ID、Age、Group 绑定在一起，比使用三个独立数组更清晰。
- **预处理逻辑**: 明白了在处理高维数据(基因矩阵)前，必须先清洗和标记低维数据(元数据)。

### 4. 下一步计划
- 读取基因表达矩阵文件。
- 根据解析好的 ID 列表，提取对应列的数据进行 Log2 变换。
