
## `sort` 命令是一个非常强大的工具，用于对文件中的行进行排序，并将结果写入标准输出。

## 1. 基本用法

### **对文件内容进行简单排序**



```Bash
sort 文件名
```

默认情况下，`sort` 命令会根据当前语言环境的整理顺序，按升序对文件的所有行进行排序。

### **将排序结果输出到新文件**

如果你想将排序后的结果保存到另一个文件，可以使用重定向：



```Bash
sort 文件名 > 新文件名
```

### **原地替换原文件内容**

如果你想用排序后的内容替换原始文件内容，可以使用 `-o` 标志：



```Bash
sort -o 文件名 文件名
```

**例如：** `sort -o names names` 会用排序后的内容替换 `names` 文件的原始内容。

## 2. 常用选项（标志）

|**选项**|**描述**|
|---|---|
|`-r`|**降序**排列 (Reverse)。|
|`-n`|**数字**排序 (Numeric sort)，按数值大小进行排序，而不是按字符串顺序。|
|`-k`|**指定排序键** (Key)。例如：`-k 2` 表示按第二个字段排序。|
|`-t`|**指定字段分隔符** (Field Separator)。例如：`-t ':'` 表示使用冒号作为字段分隔符。|
|`-b`|**忽略**每行开头处的**空白字符** (Blank)。|
|`-u`|**去重** (Unique)，只输出唯一的行。|
|`-c`|检查文件是否已排序。如果文件未排序，它将输出一个诊断信息并返回非零退出状态。|

## 3. 进阶用法：指定排序键 (`-k`)

`sort -k` 标志允许你指定按行的一部分（称为“排序键”）进行排序。

### **语法**

`-k Fstart[.Cstart][修饰符][,FEnd[.CEnd][修饰符]]`

- `Fstart`: 开始字段号。
    
- `Cstart`: 开始字段中的列号（可选）。
    
- `FEnd`: 结束字段号（可选）。
    
- `CEnd`: 结束字段中的列号（可选）。
    
- `修饰符`: 可以是其他排序选项，例如 `n` (数字排序) 或 `r` (降序)。
    

### **示例**

1. **按第二个字段排序：**
    
    
    
   ```Bash
    sort -k 2 文件名
    ```
    
2. **按第三个字段的数字大小进行降序排列：**
    
    
    
   ```Bash
    sort -k 3,3nr 文件名
    ```
    
    - `3,3`: 指定排序键从第3个字段开始，到第3个字段结束。
        
    - `n`: 数字排序。
        
    - `r`: 降序排列。
        

**注意：** 默认字段分隔符是“一个或多个连续的空白字符”（空格或制表符）。如果使用 `-t` 指定了分隔符，则分隔符本身不计为字段的一部分。
