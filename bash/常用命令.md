# Bash 常用指令速查笔记

> 符号说明：
> * `<...>` 代表需要根据实际情况替换的内容。
> * `[...]` 代表可选参数。

## 1. 导航与查看
| 指令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `pwd` | 显示当前路径 | `pwd` |
| `ls -l` | 列出详细信息 (List) | `ls -l` |
| `ls -a` | 显示隐藏文件 | `ls -a` |
| `cd <路径>` | 切换目录 (Change Directory) | `cd /home/user/downloads` |
| `cd ..` | 返回上一级目录 | `cd ..` |

## 2. 文件操作
| 指令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `touch <文件名>` | 创建空文件 | `touch notes.txt` |
| `mkdir <目录名>` | 创建文件夹 (Make Dir) | `mkdir my_project` |
| `cp <源> <目标>` | 复制文件 (Copy) | `cp file.txt file_backup.txt` |
| `cp -r <源> <目标>`| 复制文件夹 | `cp -r src/ dest/` |
| `mv <源> <目标>` | 移动或重命名 (Move) | `mv old_name.txt new_name.txt` |
| `rm <文件名>` | 删除文件 (Remove) | `rm useless_file.txt` |
| `rm -rf <目录名>` | **强制删除**文件夹及内容 | `rm -rf old_project/` (慎用) |

## 3. 查看内容
| 指令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `cat <文件名>` | 显示全部内容 | `cat script.sh` |
| `less <文件名>` | 分页查看 (按q退出) | `less large_log.log` |
| `head -n <N> <文件名>`|看前 N 行 | `head -n 5 data.csv` |
| `tail -f <文件名>` | **实时监控**文件末尾更新 | `tail -f server.log` |

## 4. 搜索与查找
| 指令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `grep "<词>" <文件>` | 在文件中搜内容 | `grep "error" app.log` |
| `find <路径> -name "<名>"`| 按文件名查找文件 | `find . -name "*.sh"` |

## 5. 权限与系统
| 指令              | 描述            | 示例                |
| :-------------- | :------------ | :---------------- |
| `chmod +x <文件>` | 赋予执行权限        | `chmod +x run.sh` |
| `sudo <命令>`     | 以管理员身份运行      | `sudo apt update` |
| `top`           | 查看系统进程 (按q退出) | `top`             |
