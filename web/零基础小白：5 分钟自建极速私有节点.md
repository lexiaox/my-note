## 🚀 零基础小白：5 分钟自建极速私有节点

本教程采用 2025 年最先进、最安全的 **VLESS + REALITY** 协议，抗封锁能力强，且能跑满 VPS 带宽。

### 第一步：准备工作 (VPS 购买与登录)

1. **购买 VPS**：推荐 Vultr、搬瓦工或 RackNerd（建议选择新加坡、日本或美国西海岸机房）。
    
2. **系统选择**：安装 **Ubuntu 22.04** 或 **Debian 12**。
    
3. **登录服务器**：
    
    - Windows 用户使用 PowerShell，Mac/Linux 使用 Terminal。
        
    - 输入：`ssh root@你的服务器IP`，然后输入密码。
        

---

### 第二步：一键安装内核与服务

登录后，直接复制并运行以下命令（建议使用官方推荐的安装脚本）：

Bash

```
# 切换为 root 用户
sudo -i

# 安装常用工具
apt update && apt install curl -y

# 执行 Xray 一键安装脚本 (官方版)
bash <(curl -Ls https://raw.githubusercontent.com/vaxilu/x-ui/master/install.sh)
```

> **注意**：安装过程中会让你设置 x-ui 面板的用户名、密码和端口（如 54321）。

---

### 第三步：面板配置 (关键步骤)

1. 浏览器访问：`http://你的服务器IP:54321`。
    
2. 进入 **“入站列表”** -> **“添加绑定”**：
    
    - **备注**：随便填。
        
    - **协议**：选择 `vless`。
        
    - **端口**：填 `443`（模拟网页标准端口）。
        
    - **传输**：`tcp`。
        
    - **流控**：`xtls-rprx-vision`。
        
3. **开启 REALITY**：
    
    - **目标网站 (dest)**：填 `www.microsoft.com:443` 或其他国外正规教育/政府网站。
        
    - **SNI**：与目标网站一致。
        
    - **PrivateKey**：点击面板上的“生成”按钮。
        
    - **ShortID**：点击面板上的“生成”按钮。
        

---

### 第四步：客户端连接

配置好后，点击操作栏里的 **“二维码”** 或 **“复制链接”**。

#### 1. 电脑端 (Clash Verge / v2rayN)

- **v2rayN**：点击“从剪贴板导入”，然后右键设为“系统代理”。
    
- **Clash Verge**：由于内核需求，建议使用 **Mihomo (Meta)** 内核，将链接转换后导入。
    

#### 2. 手机端 (v2rayNG)

- 点击右上角 `+` 号 -> **扫码导入** 或 **从剪贴板导入**。
    
- 点击连接，即可起飞。
    

---

### ⚠️ IT 大师的避坑指南 (针对兰大/教育网用户)

- **时间同步**：REALITY 协议要求服务器和手机时间误差不超过 60 秒，否则无法连接。
    
- **关闭 IPv6**：如果发现加载慢，请在客户端设置里彻底关闭 IPv6。
    
- **伪装域名**：不要选择国内能直连但内容敏感的网站，推荐使用新加坡高校（如 `www.nus.edu.sg`）或微软官网。
