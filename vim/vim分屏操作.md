在 Vim 中创建垂直分屏（将屏幕分为左右两部分），主要使用 `:vsplit` 命令。

鉴于你正在配置 `.vimrc` 且习惯 CLI 操作，这里有几种根据场景的不同用法，特别是**如何确保新窗口出现在右边**（默认有时会在左边）。

### 1. 基础命令

在 Normal 模式（按 `Esc` 进入）下输入：

- **`:vsplit`** (或简写 **`:vs`**)
    
    - 效果：将**当前文件**在右边（或左边，取决于设置）再打开一份。
        
    - 用途：同时查看同一个 C 语言源文件的开头（头文件引用）和结尾（函数实现）。
        
- **`:vsplit filename`** (或 **`:vs filename`**)
    
    - 效果：在侧边打开一个新的文件。
        
    - 用途：一边看 `.h` 头文件，一边在右边写 `.c` 实现。
