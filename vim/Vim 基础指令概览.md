
### 1. 模式切换指令

| 模式 | 作用 | 进入方式 | 退出方式 |
| :--- | :--- | :--- | :--- |
| **普通模式 (Normal)** | 移动光标、删除、复制、粘贴、执行命令。 | 任何模式下按 **`<Esc>`** 键。 | - |
| **插入模式 (Insert)** | 输入和编辑文本。 | 在普通模式下按 **`i`、`a`、`o`** 等。 | 按 **`<Esc>`** 键返回普通模式。 |
| **命令行模式 (Command-line)** | 执行保存、退出、查找替换等命令。 | 在普通模式下按 **`:`** 或 **`/`**。 | 按 **`<Esc>`** 键或回车执行命令后返回普通模式。 |

### 2. 保存与退出指令 (命令行模式)

在普通模式下按 **`:`** 后输入：

| 命令 | 作用 |
| :--- | :--- |
| `:w` | **保存**当前文件。 |
| `:q` | **退出** Vim 窗口。 |
| `:wq` 或 `ZZ` | **保存并退出**。 |
| `:q!` | **强制退出**，不保存更改。 |
| `:x` | **保存并退出**（仅在有更改时才保存）。 |

### 3. 光标移动指令 (普通模式)

| 指令 | 作用 |
| :--- | :--- |
| `h` `j` `k` `l` | 左/下/上/右移动一个字符/行。 |
| `w` | 移动到下一个单词的**开头**。 |
| `b` | 移动到上一个单词的**开头**。 |
| `0` | 移动到行首。 |
| `$` | 移动到行尾。 |
| `gg` | 移动到文件的**第一行**。 |
| `G` | 移动到文件的**最后一行**。 |

### 4. 插入模式进入指令 (普通模式)

| 指令  | 作用                  |
| :-- | :------------------ |
| `i` | 在光标**当前位置**前插入。     |
| `a` | 在光标**当前位置**后插入。     |
| `o` | 在光标所在行**下方**新开一行插入。 |
| `I` | 在当前行的**行首**插入。      |
| `A` | 在当前行的**行尾**插入。      |
| `O` | 在光标所在行**上方**新开一行插入。 |

### 5. 编辑/删除/复制指令 (普通模式)

| 指令 | 作用 | 组合示例 | 作用 |
| :--- | :--- | :--- | :--- |
| `x` | 删除光标下的字符。 | `dw` | 删除从光标到下一个单词。 |
| `d` | 删除操作符。 | `d$` | 删除从光标到行尾的内容。 |
| `y` | 复制/“拽取”操作符 (yank)。 | `yy` | 复制当前一整行。 |
| `p` / `P` | 粘贴。 | `3yy` | 复制 3 行。 |
| `u` | 撤销上一个操作 (undo)。 | `Ctrl + r` | 重做被撤销的操作 (redo)。 |
