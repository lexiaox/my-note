

`:e` 是 Vim 中在当前窗口加载文件的核心命令。

## 1. 基础用法
| 命令 | 作用 | 备注 |
| :--- | :--- | :--- |
| `:e filename.c` | 打开文件 | 支持 `<Tab>` 补全路径 |
| `:e newfile.c` | 新建文件 | 需 `:w` 才会保存到硬盘 |
| `:e .` | 打开文件管理器 | 浏览当前目录 |
| `:e /path/to/file`| 打开绝对路径文件 | |

## 2. 进阶技巧
* **强制重载 (放弃修改)**：
    ```vim
    :e!
    ```
    *适用场景*：代码改乱了想重来，或者 `git pull` 后文件在外部变更了。

* **快速编辑配置**：
    ```vim
    :e $MYVIMRC
    ```
    *适用场景*：随时随地修改 `init.lua` / `.vimrc`。

## 3. 与分屏的区别
* `:e`：**替换**当前窗口的内容（原文件进入后台 Buffer）。
* `:sp` / `:vs`：**分屏**打开新文件（原文件保持可见）。
