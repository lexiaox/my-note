
### 快捷键切换窗口

在 Vim 中，所有窗口切换命令都需要以 **`<C-w>`** (即按下 **Ctrl** 键不放，再按 **w** 键) 开头，然后紧接着输入一个方向键或字母。

| 操作 | 快捷键 | 作用 |
| :--- | :--- | :--- |
| **通用切换** | `<C-w> w` | 循环切换到下一个窗口。 |
| **向上切换** | `<C-w> k` | 切换到**上方**的窗口。 |
| **向下切换** | `<C-w> j` | 切换到**下方**的窗口。 |
| **向左切换** | `<C-w> h` | 切换到**左边**的窗口。 |
| **向右切换** | `<C-w> l` | 切换到**右边**的窗口。 |
| **切换到上一个访问的窗口** | `<C-w> p` | 切换到上一次操作所在的窗口。 |

### 窗口操作命令

| 命令 | 作用 |
| :--- | :--- |
| `:q` / `:close` | 关闭当前窗口。 |
| `:only` / `<C-w> o` | 仅保留当前窗口，关闭所有其他窗口。 |
| `<C-w> =` | 让所有窗口大小均匀分配。 |


## ⚠️ 特殊情况：右边的终端窗口 (Terminal Trap)

我看你右边运行的是 zsh。在 Vim 的终端模式（Terminal Mode）下，按键通常会被直接发送给 Shell（比如按 Ctrl+w 可能会在 Shell 里执行“删除前一个单词”的操作，而不是切换 Vim 窗口）。

如果你被“困”在右边的终端里切不出来：

先按 Ctrl + \ (反斜杠)，松开，再迅速按 Ctrl + n。

这一步的作用是将终端从“输入模式”切换回 Vim 的“普通模式”。

现在你可以正常使用 Ctrl + w + h 切回左边了。
