

所有调整窗口大小的命令都需要在 **普通模式 (Normal Mode)** 下输入。

### 1. 自动均匀分配

| 快捷键/命令 | 作用 |
| :--- | :--- |
| `<C-w> =` | 让所有窗口**大小均匀分配**（高度和宽度都平均）。 |

### 2. 增加或减小当前窗口尺寸

在命令前加数字可以指定增减的行/列数（例如：`10<C-w>+`）。

| 操作 | 快捷键 | 作用 |
| :--- | :--- | :--- |
| **增加高度** | `<C-w> +` | 将当前窗口的高度增加一行。 |
| **减小高度** | `<C-w> -` | 将当前窗口的高度减小一行。 |
| **增加宽度** | `<C-w> >` | 将当前窗口的宽度增加一列。 |
| **减小宽度** | `<C-w> <` | 将当前窗口的宽度减小一列。 |

### 3. 设置当前窗口为最大/最小尺寸

| 命令 | 作用 |
| :--- | :--- |
| `<C-w> _` | 设置当前窗口为**最大高度**。 |
| `<C-w> \|` | 设置当前窗口为**最大宽度**。 |
| `:resize 0` | 设置当前窗口为**最小高度** (1 行)。 |

### 4. 使用命令行精确调整

| 命令 | 作用 |
| :--- | :--- |
| `:resize 15` | 设置当前窗口高度为 **15 行**。 |
| `:resize +5` | 当前窗口高度**增加 5 行**。 |
| `:vert resize 40` | 设置当前窗口宽度为 **40 列**。 |
