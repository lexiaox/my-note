

### 1. 数列极限 ($\epsilon-N$)
**定义：** $\forall \epsilon > 0, \exists N \in \mathbb{Z}^+, \text{ s.t. } n > N \implies |a_n - L| < \epsilon$

**证明实例：** $\lim_{n \to \infty} \frac{2n+1}{n} = 2$
1. **分析：** $| \frac{2n+1}{n} - 2 | = \frac{1}{n} < \epsilon \implies n > \frac{1}{\epsilon}$
2. **证明：** - 任取 $\epsilon > 0$，取 $N = \lceil \frac{1}{\epsilon} \rceil$。
   - 当 $n > N$ 时，有 $n > \frac{1}{\epsilon}$，即 $\frac{1}{n} < \epsilon$。
   - 故 $| \frac{2n+1}{n} - 2 | = \frac{1}{n} < \epsilon$。
   - 证毕。

### 2. 函数极限 ($\epsilon-\delta$)
**定义：** $\forall \epsilon > 0, \exists \delta > 0, \text{ s.t. } 0 < |x - x_0| < \delta \implies |f(x) - A| < \epsilon$

**证明实例：** $\lim_{x \to 2} x^2 = 4$
1. **分析：** - 目标：$|x^2 - 4| = |x-2||x+2| < \epsilon$
   - 限制范围：设 $\delta \le 1$，则 $|x-2| < 1 \implies 1 < x < 3 \implies |x+2| < 5$
   - 放大不等式：$|x^2 - 4| < 5|x-2|$
   - 令 $5|x-2| < \epsilon \implies |x-2| < \frac{\epsilon}{5}$
2. **证明：**
   - 任取 $\epsilon > 0$，取 $\delta = \min\{1, \frac{\epsilon}{5}\}$。
   - 当 $0 < |x-2| < \delta$ 时，有 $|x+2| < 5$ 且 $|x-2| < \frac{\epsilon}{5}$。
   - 则 $|x^2-4| = |x-2||x+2| < \frac{\epsilon}{5} \cdot 5 = \epsilon$。
   - 证毕。

---

### 3. 逻辑要点 (Memory Hook)
- **寻找关系：** 证明的核心是构造出 $\delta(\epsilon)$ 或 $N(\epsilon)$ 的函数关系。
- **放大法：** 在处理多项式或三角函数时，通过先固定一个 $\delta$ 范围（如 $\delta \le 1$）来简化项。
