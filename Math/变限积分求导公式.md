# 变限积分求导公式 (微积分基本定理)

**核心概念**：处理积分上限或下限含有变量 $x$ 时的求导问题。本质是微积分基本定理与复合函数求导法则（链式法则）的结合。

## 关键公式

### 1. 基础形式
$$\left(\int_a^x f(t)dt\right)' = f(x)$$

### 2. 下限为变量
$$\left(\int_x^b f(t)dt\right)' = -f(x)$$
*推导：交换积分上下限会改变符号。*

### 3. 上限为函数 (Chain Rule)
$$\left(\int_a^{\varphi(x)} f(t)dt\right)' = f(\varphi(x))\varphi'(x)$$
*记忆口诀：代入上限 $\times$ 上限的导数。*

### 4. 通式 (上下限均为函数)
$$\left(\int_{\psi(x)}^{\varphi(x)} f(t)dt\right)' = f(\varphi(x))\varphi'(x) - f(\psi(x))\psi'(x)$$
*记忆口诀：(代入上限 $\times$ 上限导数) - (代入下限 $\times$ 下限导数)。*
