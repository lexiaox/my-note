## 物理笔记：洛仑兹变换 (Lorentz Transformation)

**洛仑兹变换**是狭义相对论的核心数学工具，用于在两个惯性参考系之间转换时间和空间坐标。它的核心目的是**在不同参考系中保持光速 $c$ 恒定**。

### 1. 核心矛盾
* **伽利略变换 (经典力学)**：认为时间是绝对的 ($t' = t$)，速度是可以简单叠加的。会导致光速在不同参考系不同，违反麦克斯韦方程组。
* **洛仑兹变换 (相对论)**：为了迎合“光速不变原理”，强行让空间和时间发生伸缩。

### 2. 关键因子：Gamma ($\gamma$)
这是一个衡量“相对论效应”强弱的系数：
$$\gamma = \frac{1}{\sqrt{1 - \frac{v^2}{c^2}}}$$
* 当 $v \ll c$ (日常低速)：$\gamma \approx 1$ (回归牛顿力学)。
* 当 $v \to c$ (接近光速)：$\gamma \to \infty$ (时间停止，空间压扁)。

### 3. 变换公式
假设参考系 $S'$ 相对于 $S$ 以速度 $v$ 沿 $x$ 轴运动：

| 维度 | 伽利略变换 (旧) | 洛仑兹变换 (新) |
| :--- | :--- | :--- |
| **位置 ($x'$)** | $x' = x - vt$ | $x' = \gamma (x - vt)$ |
| **时间 ($t'$)** | $t' = t$ | $t' = \gamma (t - \frac{v}{c^2}x)$ |

### 4. 物理推论
1.  **时间膨胀**：运动的时钟变慢。
2.  **尺缩效应**：运动的物体在运动方向上变短。
3.  **时空纠缠**：时间 $t'$ 的计算依赖于位置 $x$，说明时间和空间不是独立的，而是一个整体（时空）。

### 5. C 语言伪代码理解
```c
// 伽利略：时间是全局变量
t_new = t_old; 

// 洛仑兹：时间是依赖于空间和速度的局部变量
t_new = gamma * (t_old - (v * x_old) / (c * c));
