
### 1. 结构体的基本定义

结构体允许将不同数据类型的变量组合成一个单一的复合类型。

#### 📌 格式一：基本定义

```C
struct 标签 {
	数据类型 成员1;
	数据类型 成员2;
	// ...
};
```

#### 📌 格式二：使用 `typedef` 定义别名 (推荐)

使用 `typedef` 可以省略后续声明变量时重复写 `struct 标签`。

```C
typedef struct 标签 {
	数据类型 成员1;
	数据类型 成员2;
} 别名; 
```

### 2. 结构体的变量声明与初始化

|**操作**|**格式 (以 StudentInfo 为例)**|
|---|---|
|**声明变量**|`StudentInfo s1;`|
|**声明并初始化 (顺序)**|`StudentInfo s2 = {101, "Alice", 95.5};`|
|**声明并初始化 (指定成员)**|`StudentInfo s3 = {.name = "Bob", .id = 102};`|

### 3. 结构体的核心操作

#### 3.1 成员访问（点运算符 `.`）

使用点运算符（`.`）来访问结构体变量的内部成员。

```C
StudentInfo s1;
s1.id = 101;
printf("ID: %d\n", s1.id);
```

#### 3.2 结构体指针访问（箭头运算符 `->`）

如果有一个指向结构体的指针，使用箭头运算符（`->`）来访问其成员。



```C
StudentInfo *ptr_s = &s1;
ptr_s->id = 101; 
// 等价于：(*ptr_s).id = 101;
```

#### 3.3 赋值与复制

**结构体可以直接赋值：**


```C
// s1 的所有成员值都会复制给 s2
s2 = s1; 
```

### 4. 结构体与数组/字符串

#### ⚠️ 字符串操作注意事项

字符串成员不能直接使用 `=` 赋值，必须使用 `strcpy()` 或 `strncpy()`。


```C
#include <string.h>
StudentInfo s1;
// 正确操作：
strcpy(s1.name, "Alice"); 

// 错误示范：
// s1.name = "Alice"; // ❌ 错误：不能直接赋值数组
```
