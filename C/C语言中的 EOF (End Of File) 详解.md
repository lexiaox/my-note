
## 1. 核心概念
* **全称**：End Of File（文件结束）。
* **定义**：在 `<stdio.h>` 中定义为 `#define EOF (-1)`。
* **本质**：它不是一个字符，而是系统返回的一个**状态值**，表示“没有更多数据可读”。

## 2. 为什么 scanf 要和 EOF 配合？
`scanf` 函数的返回值决定了循环是否继续：
* **读取成功**：返回成功读取的变量个数（如 1, 2...）。
* **读取结束**：返回 `EOF` (-1)。

~~~c
int x;
// 当读不到数据时，scanf 返回 -1，循环条件不成立，退出
while (scanf("%d", &x) != EOF) {
    // 处理 x
}
~~~

## 3. 如何在控制台触发 EOF？
控制台输入是“无限流”，必须手动发送终止信号。
* **Windows**: 输入完数据后，换行，按 `Ctrl + Z`，再按回车。
* **Linux/macOS**: 按 `Ctrl + D`。

## 4. 常见技巧 (按位取反)
利用 `-1` 的补码特性简化代码：
~~~c
// ~(-1) 结果为 0，循环结束
while (~scanf("%d", &x)) {
    // ...
}
~~~
