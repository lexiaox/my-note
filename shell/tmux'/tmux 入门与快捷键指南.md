

`tmux` (Terminal Multiplexer) 允许你在一个终端窗口内创建和管理多个会话 (Sessions)、窗口 (Windows) 和面板 (Panes)。即使 SSH 连接断开，会话也能保持运行。

## 一、基本概念

| 概念               | 含义                      | 作用                            |
| :--------------- | :---------------------- | :---------------------------- |
| **Session (会话)** | 顶层容器，可以包含多个窗口。          | 独立的工作区，可以随时断开/重新连接。           |
| **Window (窗口)**  | 类似于浏览器标签页，一个会话中可以有多个窗口。 | 组织不同的任务（如：一个窗口用于编程，一个用于 Git）。 |
| **Pane (面板)**    | 窗口内的水平或垂直分割区域。          | 实现分屏效果（如：左侧 Vim，右侧 Shell）。    |

## 二、基础操作命令

| 命令                            | 作用                   |
| :---------------------------- | :------------------- |
| `tmux`                        | 启动一个新的 `tmux` 会话。    |
| `tmux attach`  (tmux a)       | 重新连接（恢复）上次断开的会话。     |
| `tmux new -s <name>`          | 创建一个命名的新会话。          |
| `tmux ls`                     | 列出所有正在运行的 `tmux` 会话。 |
| `tmux kill-session -t <name>` | 杀死指定的会话。             |

## 三、核心快捷键 (Prefix Key: `Ctrl+b`)

所有 `tmux` 快捷键都必须先按 **Prefix Key** (默认是 `Ctrl+b`，即按住 Ctrl 键和 b 键)，然后松开，再按下一个键。

### 1. 面板操作 (Pane)

| 快捷键                | 作用                           |     |
| :----------------- | :--------------------------- | --- |
| `Ctrl+b` + **`%`** | **垂直**分割当前面板（实现左右分屏）。        |     |
| `Ctrl+b` + **`"`** | **水平**分割当前面板（实现上下分屏）。        |     |
| `Ctrl+b` + **方向键** | 切换到相邻的面板。                    |     |
| `Ctrl+b` + **`z`** | 切换当前面板的**最大化**和恢复状态。         |     |
| `Ctrl+b` + **`x`** | 关闭当前面板。                      |     |
| `Ctrl+b` + **`[`** | 进入**复制模式**（用 Vim 键位滚动和复制文本）。 |     |
| `Ctrl+b` + **`]`** | 粘贴复制模式中复制的文本。                |     |

### 2. 窗口操作 (Window)

| 快捷键                      | 作用            |
| :----------------------- | :------------ |
| `Ctrl+b` + **`c`**       | 创建一个新的窗口。     |
| `Ctrl+b` + **`n`**       | 切换到下一个窗口。     |
| `Ctrl+b` + **`p`**       | 切换到上一个窗口。     |
| `Ctrl+b` + **`,`**       | 重命名当前窗口。      |
| `Ctrl+b` + **`0` 到 `9`** | 快速切换到编号对应的窗口。 |

### 3. 会话操作 (Session)

| 快捷键                | 作用                            |
| :----------------- | :---------------------------- |
| `Ctrl+b` + **`d`** | **断开** (Detach) 当前会话，让它在后台运行。 |
| `Ctrl+b` + **`s`** | 列出所有会话，并允许你选择切换。              |
