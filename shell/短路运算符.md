| **符号**   | **名字** | **口诀**      | **场景**                    |
| -------- | ------ | ----------- | ------------------------- |
| **`&&`** | AND    | **成功了才...** | 也就是 `Then`。前一步必须成功。       |
| **`      |        | `**         | OR                        |
| **`;`**  | 分号     | **做完A做B**   | 也就是 `Next`。单纯的顺序执行，不关心对错。 |
