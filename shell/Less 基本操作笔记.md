

### 一、启动与退出

| **命令**            | **功能** | **说明**                |
| ----------------- | ------ | --------------------- |
| **`less <文件名>`**  | 启动     | 用 `less` 打开一个文件进行查看。  |
| **`q`** 或 **`Q`** | 退出     | 退出 `less` 程序，返回命令行界面。 |
| **`h`** 或 **`H`** | 帮助     | 显示 `less` 命令的帮助界面。    |

---

### 二、移动与导航 (滚动)

| **命令**                       | **功能** | **说明**           |
| ---------------------------- | ------ | ---------------- |
| **`j`** 或 **`↓`**            | 向前移动   | 向下滚动 **一行**。     |
| **`k`** 或 **`↑`**            | 向后移动   | 向上滚动 **一行**。     |
| **`Ctrl + f`** 或 **`Space`** | 向前翻页   | 向下滚动 **一屏**（一页）。 |
| **`Ctrl + b`** 或 **`b`**     | 向后翻页   | 向上滚动 **一屏**（一页）。 |
| **`d`** 或 **`Ctrl + d`**     | 向前滚动   | 向下滚动 **半屏**。     |
| **`u`** 或 **`Ctrl + u`**     | 向后滚动   | 向上滚动 **半屏**。     |

---

### 三、文件内快速跳转

| **命令**            | **功能**  | **说明**                    |
| ----------------- | ------- | ------------------------- |
| **`g`** 或 **`<`** | 跳转到文件开头 | 跳转到文件的**第一行**。            |
| **`G`** 或 **`>`** | 跳转到文件末尾 | 跳转到文件的**最后一行**。           |
| **`<数字>g`**       | 跳转到指定行  | 跳转到文件中的**第 <数字> 行**。      |
| **`m<字母>`**       | 标记当前位置  | 使用一个小写字母（例如 `ma`）来标记当前位置。 |
| **`'<字母>`**       | 跳转到标记位置 | 跳转到之前标记的位置（例如 `'a`）。      |

---

### 四、搜索操作

| **命令**        | **功能** | **说明**                    |
| ------------- | ------ | ------------------------- |
| **`/ <关键词>`** | 向前搜索   | **向下**（文件末尾方向）搜索 `<关键词>`。 |
| **`? <关键词>`** | 向后搜索   | **向上**（文件开头方向）搜索 `<关键词>`。 |
| **`n`**       | 下一个匹配  | 重复上一次搜索，跳到**下一个**匹配项。     |
| **`N`**       | 上一个匹配  | 重复上一次搜索，跳到**上一个**匹配项。     |

---

### 五、其它实用功能

| **命令**        | **功能**  | **说明**                                                                                |
| ------------- | ------- | ------------------------------------------------------------------------------------- |
| **`F`**       | 实时追踪    | 进入类似 `tail -f` 的模式，**实时查看**文件末尾新增的内容（常用于日志）。<br><br>  <br><br>**按 `Ctrl + c` 退出此模式。** |
| **`v`**       | 用编辑器打开  | 使用系统默认编辑器（通常是 `vi` 或 `vim`）打开当前正在查看的文件进行编辑。                                           |
| **`:` + `n`** | 查看下一个文件 | 如果您使用 `less file1 file2` 启动了多个文件，此命令用于查看列表中的下一个文件。                                    |
| **`=`**       | 显示状态    | 在屏幕底部显示当前文件名、行号、文件大小等信息。                                                              |
