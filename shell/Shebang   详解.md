
## 定义
位于脚本文件**第一行**的 `#!` 符号，用于指定脚本的解释器。

## 作用
告诉操作系统用哪个程序来执行这个脚本文件（如 Bash, Python, Perl）。

## 常用写法
| 写法                    | 含义                 | 备注                  |
| :-------------------- | :----------------- | :------------------ |
| `#!/bin/bash`         | 使用绝对路径的 Bash       | 最常见，但依赖路径固定         |
| `#!/usr/bin/env bash` | **推荐**：在环境中查找 Bash | 移植性更强，不担心安装路径不同     |
| `#!/bin/sh`           | 使用标准 sh            | 兼容性好，但不支持 Bash 特有语法 |
| `#!/usr/bin/python3`  | 使用 Python 3        | 运行 Python 脚本        |

## 注意事项
1. **位置严格**：必须在文件的 **第1行，第1列**。哪怕前面有一个空格都会失效。
2. **本质**：只有通过 `./script.sh` 方式运行时才生效；如果显式运行 `bash script.sh`，Shebang 会被忽略。
