

### 1. 泰勒公式的结构

$$f(x) = P_n(x) + R_n(x)$$

其中 $P_n(x)$ 是泰勒多项式， $R_n(x)$ 是余项（误差）。

---

### 2. 推导泰勒多项式 $P_n(x)$ 的系数

**核心要求：** 使 $P_n^{(k)}(x_0) = f^{(k)}(x_0)$ (对于 $k=0, 1, \dots, n$)。

1.  **多项式形式：**
    $$P_n(x) = c_0 + c_1(x-x_0) + c_2(x-x_0)^2 + \dots + c_n(x-x_0)^n$$

2.  **确定系数 $c_k$：**
    对 $P_n(x)$ 求 $k$ 阶导数并代入 $x=x_0$，得到：
    $$P_n^{(k)}(x_0) = k! c_k$$
    因此，系数 $c_k$ 必须满足：
    $$c_k = \frac{f^{(k)}(x_0)}{k!}$$

**泰勒多项式：**
$$P_n(x) = \sum_{k=0}^{n} \frac{f^{(k)}(x_0)}{k!}(x-x_0)^k$$

---

### 3. 余项 $R_n(x)$ 的推导 (拉格朗日形式)

**方法：** 通过对微积分基本定理的表达式重复使用**分部积分法**来确定余项的精确形式。

1.  **起点：** $f(x) - f(x_0) = \int_{x_0}^{x} f'(t) dt$

2.  **分部积分 $n$ 次，得到积分余项：**
    $$R_n(x) = \int_{x_0}^{x} \frac{(x-t)^n}{n!} f^{(n+1)}(t) dt$$

3.  **应用积分中值定理，得到拉格朗日余项：**
    $$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} (x-x_0)^{n+1} \quad (\xi \text{ 介于 } x_0 \text{ 和 } x \text{ 之间})$$

---

### 4. 余项 $R_n(x)$ 的作用

| 余项形式 | 主要用途 | 关键价值 |
| :--- | :--- | :--- |
| **拉格朗日** | 误差估计、精度控制。 | 提供了**误差上界**，确保数值计算的准确性。 |
| **皮亚诺 $o((x-x_0)^n)$** | 求极限、无穷小比较、定阶。 | 简化计算，确保**理论严谨性**。 |
| **极限 $\to 0$** | 泰勒级数收敛性判断。 | 确定函数能否展开为**无穷级数**。 |
