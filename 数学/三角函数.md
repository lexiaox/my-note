
|   函数   |       别名       |  与基本函数的关系  |
| :----: | :------------: | :--------: |
| sin(θ) |   正弦 (Sine)    | 1 / csc(θ) |
| cos(θ) |  余弦 (Cosine)   | 1 / sec(θ) |
| tan(θ) |  正切 (Tangent)  | 1 / cot(θ) |
| csc(θ) | 余割 (Cosecant)  | 1 / sin(θ) |
| sec(θ) |  正割 (Secant)   | 1 / cos(θ) |
| cot(θ) | 余切 (Cotangent) | 1 / tan(θ) |
