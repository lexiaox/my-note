# 1. if 语句

-  在 if 后面要加 **:** 表示程序块区域 ，之后同一区域范围要有相同的缩进，建议为4个空格，空格和 **Tab** 不可混用 
-  允许 `8<=a<=9` 的表达
-  e.g.
```c
import sys

odds = [arg for arg in sys.argv[1:] if int(arg) % 2]
print(odds)
```
 只有 if 条件式成立是，for 左边的表达式才会被执行
```c
  def max(num1 ,num2 ) :
	return num1 if num1 > num2 else num2 
```
如果成立运行左边，否则运行右边
