
### 渲染后的文本

| **元素**      | **示例**                                                      | **说明**                                   |
| ----------- | ----------------------------------------------------------- | ---------------------------------------- |
| **标题**      | # 一级标题<br><br>  <br><br>## 二级标题<br><br>  <br><br>### 三级标题   | Markdown 支持六级标题，但 Obsidian/常见编辑器常用一级到三级。 |
| **粗体**      | **粗体文本**                                                    | 用于强调重要内容。                                |
| _斜体_        | _斜体文本_                                                      | 用于表示着重、书名等。                              |
| **_粗斜体_**   | **_粗斜体文本_**                                                 | 同时强调和着重。                                 |
| ~~删除线~~     | ~~删除线文本~~                                                   | 表示内容已过时或被删除。                             |
| `行内代码`      | `print("Hello, world!")`                                    | 用于表示代码片段、命令、文件路径等。                       |
| **列表 (无序)** | * 项目 1<br><br>  <br><br>* 项目 2<br><br>  <br><br>    * 子项目 A | 使用 `*`、`+` 或 `-` 创建无序列表，通过缩进创建子列表。       |
| **列表 (有序)** | 1. 第一项<br><br>  <br><br>2. 第二项<br><br>  <br><br>    1. 子项 i | 使用数字后跟 `.` 创建有序列表，数字不影响最终顺序。             |
| **任务列表**    | - [x] 已完成任务<br><br>  <br><br>- [ ] 待办任务                     | Obsidian 或支持 Task List 的 Markdown 编辑器常用。 |
| **链接**      | [我的博客](https://example.com/)                                | 创建超链接。                                   |
| **图片**      |                                                             | 插入图片。                                    |
| **引用块**     | > 这是引用内容                                                    | 用于引用他人的话或特定段落。                           |
| **水平线**     | ---                                                         | 用于分隔内容，如段落或主题切换。                         |
| **代码块**     | `python<br>def func():<br>&nbsp;&nbsp;&nbsp;&nbsp;pass<br>` | 用于展示多行代码或脚本，支持语法高亮（如 `python`）。          |
| **表格**      | 表格头 1                                                       | 表格头 2<br><br>  <br><br>:-                |
